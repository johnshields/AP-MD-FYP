# Run the Project

### Repota App
### Open the repository directory in Command-Line and enter:
```bash
$ cd repota/repotaApp
$ ionic serve
```

### Horton Backend
### Open the repository directory in Command-Line and enter:
```bash
$ cd horton
$ go build && go run main.go
```
